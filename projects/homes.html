Homes.com

Agent Client Connection

Brand/Project:

CoStar Group, Homes.com, Residential Real Estate

Role:

UX Designer

Date:

Q3-Q4 2022

Context:

In 2022 CoStar Group was branching out into residential real estate with their Homes.com product. We needed to create designs for the interactions between real estate buyer agents and their buyer clients utilizing the existing Homes.com design system.

The Problem

How do we create a product that facilitates and enhances the relationship between a buyer agent and their buyer client?

* Additional Considerations:

  • Agents can accumulate hundreds of clients over the course of their careers
  • Because this is a new product there is a need to acquire and retain as many agent and consumer users as possible to build the user base

The Solution

Outline the interaction requirements starting from the most general proceeding to the most specific.

Users

Agent: Ann Taylor

  • Has been a real estate agent for 20+ years
  • Acts mostly on the buyer side
  • Worked with hundreds of clients over the course of her career
  • Has about 6 active clients in different stages of buying
  • Not technologically savvy
Consumer: Rosemarie Townsend
  • Married with two children
  • First time home buyer
  • Enjoys shopping/browsing homes for sale online
  • Connected with Ann Taylor at an open house

Requirements Outline

Agent:

  • Connect
    • Send Invitation
      • Email
      • QR Code

  • Pending Client List
    • View pending invitations
    • Accept/reject Invitation from Consumer

  • Manage Connections
    • View Active Client List
      • Pin Active Clients
      • Remove Clients

Consumer:

  • Connect
    • Send Invitation
      • To registered Agent
      • To unregistered Agent

  • Pending Client List
    • View pending invitations
    • Accept/reject Invitation from Agent

  • Manage Connections
    • View Agent
      • Remove Agent

Actions Flows

Agent to Consumer

  • Agents need a way to distinguish between active and inactive clients so the team discussed and agreed on a pinning functionality

  • This needed to plug into additional agent/client interactions, we built out an agent view of the consumer profile so agents could see their client's favorite homes

Consumer to Agent

  • For an agent to create a Homes.com account they must be registered with an MLS. If an agent did not have an account, they must verify that registration

  • For this early version, consumers were limited to one agent connection
